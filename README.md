# Hệ thống Quản lý Bếp Trung Tâm và Cửa hàng Franchise

Central Kitchen and Franchise Store Management System - Admin Panel

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm run dev
```

## Cấu trúc chức năng

- **Layout**: Sidebar (thu gọn được), Header, Routing
- **Quản lý Users**: CRUD, gán role (admin/manager/staff/viewer), thêm/xóa user
- **Quản lý Cửa hàng & Bếp**: Quản lý cửa hàng franchise và bếp trung tâm
- **Cấu hình hệ thống**: Cài đặt chung, bảo mật, thông báo

## Công nghệ

- React 19 + Vite
- React Router DOM
- React Icons
